# Anleitung: Wie erstelle ich eine Wiki-Page?
Das hier ist eine kurze step by step Anleitung, um korrekt eine Wiki-Page einzurichten.

## Step 1
Jedes erstellte Repo wird standardmässig keine Wiki haben. Deshalb muss dies in den Einstellungen aktiviert werden.
Navigiere auf **"Settings"**.

![image](../mediafiles/images/scr-navigate-settings.png)

## Step 2
Navigiere in der linken Sidebar auf **"General"**. Danach im Abschnitt **"Features"** einen Haken bei **"Wiki"** setzen.

![image](../mediafiles/images/scr-navigate-wiki-enable.png)

## Step 3
Da nun der Abschnitt **"Wiki"** im Reiter angezeigt wird, kann nun darauf navigiert werden. 
Im Wiki drinnen kann durch den Button **"New page"** eine neue Seite erstellt werden.

![image](../mediafiles/images/scr-navigate-new-page.png)

## Step 4
In der **"Create new page"**-Maske kann nun der Inhalt ausfüllt werden, um eine Wiki-Page zu erstellen.
Hier ist der Edit-Mode **"Markdown"** empfehlenswert. Dies da Markdown noch benutzerfreundlich ist.
*Tipp: Unter den Abschnitt **Preview** kann die aktuelle Ansicht von Inhalt anzeigt werden*

![image](../mediafiles/images/scr-create-page.png)

## Step 5
**Nun hast du es geschafft! Deine erste Wiki-Page.**

![image](../mediafiles/images/scr-result.png)
